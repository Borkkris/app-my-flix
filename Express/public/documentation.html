myFlix Documentation

Introduction

This is an REST API for application called myFlix that interacts with a database and stores data about different movies.
You will be able to Sing up,update your personal information,and create a list of your favorite movies.

Allow new users to register /users POST

JSON object with data about user. Required: "Username"

Example:

{
"Username": "Mary",
"Password": "test1234",
"Email": "Mary@gmail.com",
"Birthday": "01/01/1990"
}

An object containing data about specific director. Every key is described with value of a string.

Example:

{
"Username": "Mary",
"Password": "test1234",
"Email": "Mary@gmail.com",
"Birthday": "1989-12-31T23:00:00.000Z",
"FavoriteMovies": [],
"_id": "62fd0baf1b328e6f719f03a8",
"__v": 0
}

Response status codes:

201: OK- JSON object

500: Internal server error- Something went wrong
Allow users to update their user info (username, password, email, date of birth) by Username /users/:Username PUT

JSON object with data about user and the changed value. (i.e. Password) Required: "Username"

Example:

{
"Username": "Mary",
"Password": "test1234", (changed to "1234"!)

"Email": "Mary@gmail.com",
"Birthday": "01/01/1990"
}

JSON object with data about user which includes all informations and the updated value.

Example:

{
"Username": "Mary",
"Password": "1234", (updated Password!)
"Email": "Mary@gmail.com",
"Birthday": "01/01/1990"
}

Response status codes:

500: Internal server error- Something went wrong
Allow users to add a movie to their list of favorites /user/:Username/movies/:movieID POST

JSON object with updated list of movies in Favorite Movies array.

"FavoriteMovies": [
"62fbc84a0b9b20397a3e7478"
],
(i.e. this MovieID belongs to the Movie "Joker")

Response status codes:

500: Internal server error- Something went wrong
Allow users to remove a movie from their list of favorites /users/:Username/movies/:movieID DELETE

A JSON with users information

Example:

{
"_id": "62fd0baf1b328e6f719f03a8",
"Username": "Mary",
"Password": "1234",
"Email": "Mary@gmnail.com",
"Birthday": "1989-12-31T23:00:00.000Z",
"FavoriteMovies": [], (MovieID removed)
"__v": 0
}

Response status codes:

500: Internal server error- Something went wrong
Allow existing users to deregister /users/:Username DELETE

Expected (HTML) response data: A confirmation message that users account has been deleted.

Example:

"Mary has been deleted"

Response status codes:

200: OK- Users account has been deleted

400: Bad request- Users name was not found

500: Internal server error- Something went wrong
