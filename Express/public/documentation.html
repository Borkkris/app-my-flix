myFlix Documentation

Introduction

This is an REST API for application called myFlix that interacts with a database and stores data about different movies.
You will be able to Sing up,update your personal information,and create a list of your favorite movies.

Business Logic URL HTTP Method Request body data format Response body data format
Return a list of ALL movies to the user /movies GET

An array with list of movies. Each object contains data about movies. Every key is described with value of a string.

Example:

[
{
"Title":"Hot Rod",
"Description":"The film stars Andy Samberg as amateur stuntman Rod Kimble, whose stepfather, Frank (Ian McShane), continuously mocks and disrespects him. When Frank becomes ill, Rod raises money for his heart operation by executing his largest stunt yet.",
"Genre": {
"Name":"Action/Comedy",
"Description":"A film or television programme which blends comedy with a lively plot and fast-paced action."
},
"Director": {
"Name":"Akiva Schaffer",
"Bio":"Akiva Schaffer was born on December 1, 1977 in Berkeley, California, USA. He is a writer and director, known for Popstar: Never Stop Never Stopping (2016), Saturday Night Live (1975) and The Lego Movie (2014). He has been married to Liz Cackowski since 2010. They have two children."
},
"ImageURL":"https://www.pastposters.com/cw3/assets/product_full/R/hot-rod-cinema-quad-movie-poster-(3).jpg",
"Year":"2007"

Response status codes:

200: OK

500: Internal server error- Something went wrong
Return data (description, genre, director, image URL, whether it's featured or not) about a single movie by title to the user /movies/genre/:genreName GET

An object containing data about specific movie. Every key is described with value of a string.

Example:

{
"Title":"Zoolander",
"Description":"At the end of his career, a clueless fashion model is brainwashed to kill the Prime Minister of Malaysia.",
"Genre": {
"Name":"Action/Comedy",
"Description":"A film or television programme which blends comedy with a lively plot and fast-paced action."
},
"Director": {
"Name":"Ben Stiller",
"Bio":"Benjamin Edward Meara Stiller (born November 30, 1965) is an American actor, comedian, producer, director, and screenwriter. He is the son of the comedians and actors Jerry Stiller and Anne Meara."
},
"ImageURL":"https://encrypted-tbn2.gstatic.com/images?q=tbn:ANd9GcQ28jpUvklqVl9xk9sHzTn2WnYQN026nMZLYpeZsHGKxDCrQDRi",
"Year":"2001"
},

Response status codes:

201: OK- JSON object

500: Internal server error- Something went wrong
Return data about a genre (description) by name/title (e.g., “Thriller”) /movies/:title GET

An object containing data about specific genre. Every key is described with value of a string.

Example:

{
"Name":"Comedy",
"Description":"Comedies are light-hearted dramas, crafted to amuse, entertain, and provoke enjoyment."
},

Response status codes:

200: OK- JSON object

400: Bad request- Genre not found!

500: Internal server error- Something went wrong
Return data about a director (bio, birth year, death year) by name /movies/directors/:directorName GET

An object containing data about specific director. Every key is described with value of a string.

Example:

{
"Name":"Ben Stiller",
"Bio":"Benjamin Edward Meara Stiller (born November 30, 1965) is an American actor, comedian, producer, director, and screenwriter. He is the son of the comedians and actors Jerry Stiller and Anne Meara."
},

Response status codes:

200: OK- JSON object

400: Bad request- Director not found!

500: Internal server error- Something went wrong
Allow new users to register /users POST

JSON object with data about user. Required: "name"

Example:

{
"Name": "Mary",
}

JSON object with data about user which includes automatic generated ID

Example:

{
"Name": "Mary",
"Email": "mary@email.com,"
"Password": "password123,"
"favoriteMovies": [],
"id": "8a757bb9-2eec-46bf-86f0-443ff742cf3a"
}

Response status codes:

201: Created- New user has been added

400: Bad request- User already exists
Allow users to update their user info (username, password, email, date of birth) by id /users/:id PUT

JSON object with data about user which includes,"id" with integer value, "Name", and "favorite movies" with an array.

Example:

{
"Name": "Melissa",
"Email": "mary@email.com",
"Password": "password123",
"favoriteMovies": [],
"id": 8a757bb9-2eec-46bf-86f0-443ff742cf3a,
}

Response status codes:

200: OK- New user has been added

500: Internal server error- Something went wrong
Allow users to add a movie to their list of favorites /users/:id/:movieTitle POST

JSON object with updated list of movies in Favorite Movies array.

{ Hot Rod has been added to Melissa's array }

Response status codes:

200: OK- Updated users array list

500: Internal server error- Something went wrong
Allow users to remove a movie from their list of favorites /users/:id/:title DELETE

A message indicating the movie has been added to the user's array.

Example:

{ Hot Rod has been removed from Melissa's array }

Response status codes:

200: OK- Movie has been removed from users list!

500: Internal server error- Something went wrong
Allow existing users to deregister /users/:id DELETE

Expected response data: A confirmation message that users account has been deleted.

Example:

{ "user 1 has been deleted" }

Response status codes:

200: OK- Users account has been deleted

400: Bad request- Users name was not found

500: Internal server error- Something went wrong
